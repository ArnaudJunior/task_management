# 📌 Task Management App

# 📝 Description

Task Management App est une application mobile de gestion de tâches développée en Flutter pour le frontend et Laravel pour le backend. Elle permet aux utilisateurs de planifier, suivre et organiser leurs tâches avec un calendrier interactif, un affichage des tâches filtrées par date et un écran détaillé pour chaque tâche. L'application suit une architecture propre et bien structurée pour garantir évolutivité et performance.

# 🚀 Fonctionnalités principales

1. 📅 Calendrier interactif : Sélection d'une date pour afficher les tâches correspondantes

2. ✅ Gestion des tâches : Création, modification, assignation, suivi des progrès

3. 👥 Attribution des tâches : Affichage des personnes assignées avec avatars

4. 📊 Suivi de progression : Indicateur visuel du niveau d'avancement d'une tâche

5. 🔍 Détail des tâches : Informations complètes sur la tâche avec description et délais
